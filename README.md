# Tech-Stack

* __Retrofit__ : For Network calls
* __Architecture__ : MVVM
* __Coroutines__ for background operations like fetching network response
* __Live Data__ : To notify view for change
* __Hilt__ : For dependency injection
* __Language__ : Kotlin
